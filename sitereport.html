Development Report

This report reflects my journey learning web development, design, and debugging over the term. It has been a rollercoaster of challenges and achievements, which I have documented below. I started by exploring how to structure and design a web page by using HTML and CSS. Making sure that my pages were responsive and displayed well on devices.

Reflective Discussion

Throughout this module, I have gained a deeper understanding of HTML, CSS, and the fundamentals of web design. Initially, I struggled with grasping the box model and layout positioning, but as I continued to practice, I became more comfortable with these concepts. Debugging was another challenge, particularly when dealing with layout inconsistencies across different browsers. However, the process of validating my code helped identify and fix errors.

Technical and Design Development

The website’s design evolved significantly over the term. I focused on creating a clean and cohesive user interface by choosing soft colors like #B38B59 for highlights and a neutral background color of #F8F4EC to maintain a sophisticated look. For fonts, I opted for ‘Arial’ due to its simplicity and readability. I was inspired by modern portfolio websites like Awwwards, which helped shape my approach to minimalism and user-centric design.
